# Documentation
This folder contains documentation for the specifications and decisions for Promethea. It is best opened with [Obsidian](https://obsidian.md/), as it contains files with links to other markdown files, which are nicely navigable and viewable there. 

## Content

### Software Requirements Specification (SRS)
The requirements for this project are specified in the [main SRS file](./srs.md) and the [different requirement types](./requirements/). 
The template is taken from [here](https://github.com/jam01/SRS-Template).


### Software Design Description (SDD)
The design can be found in the [main SDD file](./sdd.md), along with the [individual views](./design/).
The template is taken from [here](https://github.com/jam01/SDD-Template).

### Architectural Decision Record (ADR)
All architectural decisions can be found in the [`decisions`](./decisions/) folder. 
The template is taken from [here](https://github.com/adr/madr/).
