# REQ-PORT-001 — Supported operating environments

## Statement 
The system shall document supported server, desktop, and mobile operating environments for each release.

## Rationale 
Platform breadth is a stated nice-to-have, but exact support must be explicit to avoid false compatibility claims.

## Acceptance Criteria
- Release notes list supported server OS/container environments.
- Desktop release notes list supported desktop OSes when desktop builds exist.
- Mobile/browser support notes list supported mobile access method when mobile support exists.

## Verification Method 
Inspection
