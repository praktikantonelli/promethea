# REQ-FUNC-020 — Track reading status

## Statement
The system shall track reading status for books using defined statuses including at least unread, currently reading, read, and abandoned.

## Rationale
The Goodreads-replacement goal requires reading-status tracking.

## Acceptance Criteria
- A user can set and update a book reading status.
- The current status is visible on the book detail page and in reading views.
- Changing a status records the change time or creates a reading event.

## Verification Method
Test

## More Information
Additional statuses such as paused are TBD.
