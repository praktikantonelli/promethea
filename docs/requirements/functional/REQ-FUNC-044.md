# REQ-FUNC-044 — Fetch metadata from multiple sources

## Statement
The user shall be able to fetch metadata candidates from multiple sources and combine them into one set of metadata.

## Rationale
Multiple sources may have incomplete but complementing data.

## Acceptance Criteria
- The user can set the sources to be queried
- The system fetches from all selected sources
- The user merges the results into one set of metadata

## Verification Method
Demonstration
