# REQ-FUNC-041 — Allow rereading books

## Statement
The system shall allow a user to reread a book and still count it in the analytics.

## Rationale
Rereading a book should also be accounted for in analytics. Separating between an owned work and a read book would allow that.

## Acceptance Criteria
- The user is able to enter reading stats for the same book multiple times.

## Verification Method
Test

## More Information
Exact split between an owned book and a read book will be defined via architectural decision.
