# REQ-FUNC-036 — Sync progress between e-book and audiobook

## Statement
The system shall, in a future release, support progress synchronization between an e-book and audiobook representation of the same work.

## Rationale
The draft lists progress sync between audiobook and e-book as a potential future feature.

## Acceptance Criteria
- The system can associate e-book and audiobook editions with the same work.
- The system can store mapping metadata or conversion rules for progress positions.
- The UI can show synchronized progress where mapping is available.

## Verification Method
Analysis

## More Information
Very optional future feature; exact synchronization model is TBD.
