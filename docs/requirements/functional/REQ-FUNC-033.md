# REQ-FUNC-033 — Synchronize books to e-reader targets

## Statement 
The system shall support synchronizing selected EPUB files to configured e-reader or folder targets.

## Rationale 
The project has a nice-to-have goal of replacing parts of an e-reader cloud with automatic PC-to-e-reader syncing.

## Acceptance Criteria
- A user can select books and a configured target for sync.
- The system copies or updates selected files on the target according to target rules.
- Sync results are recorded per file.

## Verification Method 
Demonstration

## More Information
Deferred until device sync milestone; exact e-reader support is TBD.
