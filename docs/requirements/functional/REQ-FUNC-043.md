# REQ-FUNC-043 — Cherry-picking metadata

## Statement
The user shall be able to manually select only parts of a metadata candidate.

## Rationale
When metadata is fetched automatically, it is possible that not all of the gathered information is accurate or wanted.

## Acceptance Criteria
- The user fetches metadata for a book
- The fetched metadata is presented to the user
- The user can toggle whether each item of metadata shall be applied

## Verification Method
Demonstration
