# REQ-FUNC-014 — Display series detail view

## Statement
The system shall display all books belonging to a selected series.

## Rationale
Browsing by series helps finding the next book to read.

## Acceptance Criteria
- A series detail page lists all books assigned to the series.
- Series entries display ordering information when available.
- Books on the series detail page link to book detail pages.

## Verification Method
Demonstration

## More Information
Numbering/position model must support decimals.
